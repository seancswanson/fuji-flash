The Japanese beginner's toolbelt

Study smarter, have fun, learn lots!

Fuji Flash is a collection of study aides for learning the fundamentals of Japanese language. Quiz yourself using flash cards with premade decks including hiragana, katakana, numbers and common phrases.

Free forever!

Open-source web app built in Angular

This project was started by myself, Sean Swanson, in 2021 to aid in my own quest to learn Japanese and also to practice developing apps for the web. This project will remain free for all and made open-source for all to contribute if they wish. Development is ongoing and new features added from time to time.
